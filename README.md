# EnergyBot

EnergyBot is a Telegram-based gym management assistant with an admin panel built on FastAPI. It orchestrates member onboarding, subscription tracking, payments, and AI-generated workout plans through clear service layers and interfaces.

## Getting Started

1. Create and activate a Python 3.11+ virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Copy `.env.example` to `.env` and fill in the required values.
